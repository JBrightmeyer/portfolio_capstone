Portfolio Creation!

Project Summary

This application allows users to create profiles, add information (including job history, personal description, relevant links, github projects and education history) and share a public link to their portfolio to share.

Tech Information

This website uses HTML/CSS/Javascript as the front end. It utilized a combination of bootstrap, custom css and some extra libraries like Chart.js to display. The back end is written in Python/Flask with SQLAlchemy and a PostgreSQL server for data persistance. It uses Flask-Login for authentication and is hosted on Render.com

How to Use

This account is unable to be edited but the various menus and user paths are available up until the data is submitted. This is to prevent undesired additions to the account. You may also create your own account and edit as you please!
